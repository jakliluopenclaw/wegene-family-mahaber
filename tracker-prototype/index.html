How the automatic rotation works

  1. Current member chooses Host or Pass.
  2. Host date must be at least 21 days from today.
  3. If they pass, they enter the pass queue.
  4. When a scheduled hosting is confirmed complete, the system checks the pass queue first.
  5. If the pass queue is empty, normal assigned order continues.
