Rotation controls

Telegram reminders

  • Reminder to current member to host/pass
  • Reminder after date is scheduled
  • Reminder to family before gathering

Data safety

  • Export members/history/state as JSON/CSV
  • No member-facing clear-history button
  • No bot tokens or secrets in frontend files
  • Audit log for all admin actions

Admin-only data that must remain separately protected

Admin controls, Telegram chat IDs, bot tokens, private notes, delete buttons, payment admin data, audit logs, and credentials. Phone numbers may be visible inside the member-password site for MVP, but should not be exposed publicly.
